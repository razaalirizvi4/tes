# Vendor User Guide – Screenshots

Place dashboard screenshots here for the Help Center page. The app expects these filenames (all under `/guide/`):

| File | Section |
|------|--------|
| `login.png` | Getting Started – How to login |
| `restaurant-profile.png` | Getting Started – Complete restaurant profile |
| `orders-page.png` | Managing Orders – Viewing new orders |
| `accept-order.png` | Managing Orders – Accept/Reject order |
| `update-order-status.png` | Managing Orders – Update order status |
| `add-menu-item.png` | Managing Menu – Adding menu items |
| `edit-menu-item.png` | Managing Menu – Editing menu items |
| `upload-food-image.png` | Managing Menu – Uploading food images |
| `restaurant-details.png` | Restaurant Settings – Updating restaurant details |
| `opening-hours.png` | Restaurant Settings – Setting opening hours |

If a file is missing, the Help Center shows a “Screenshot placeholder” box. Use PNG (or JPG/WebP); the page uses Next.js `Image` with `unoptimized` for local files.
